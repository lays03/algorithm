# 1. 递推算法
后面问题的解可以由前面问题的解，递推而来。

用若干步可重复的简单运算来描述复杂问题的方法。

递推的特点在于，每一项和他前面的若干项有一定关联，这种关联一般可以通过递推关系式来表示，可以通过前面若干项得出某项的数据。

递推关系式 + 边界

## 1.1 n封信错排问题
递推关系式：f[1] = 0    f[2] = 1;
f[i] = (n - 1) * f[n - 1] + (n - 1) * f[n - 2]
     = (n - 1) * (f[n - 1] + f[n - 2])

## 1.2 二级递推
- 杨辉三角
```cpp
int yanghui(int n, int m)
{
    if(m == 1 || m == n) return 1;
    else{
        return yanghui(n - 1, m) + yanghui(n - 1, m - 1);
    }
}
```
